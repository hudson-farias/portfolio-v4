# v2 — `<HF/>` (aprovado na sua referência)

Baseado nos **O** da imagem:

- `<HF/>` horizontal, sem círculo / hexágono / caixa
- **H** em cor de texto (`foreground`)
- **F** e símbolos `<`, `/`, `>` em `primary` (azul no dark)

Os **X** eram logos dentro de formas (círculo, hex, quadrado).

PNG transparente em `light/` e `dark/`. Já aplicado no header + `icon.svg` / `apple-icon.svg`.

Variante alternativa que você também marcou: `{HF}` com `</>` embaixo — posso gerar se quiser comparar.
